What does this test do?

You answer short statements on a 5-point scale. At the end you get: your position on a 2D map plus a separate democratic norms score.

  • Estimated time: 3 to 6 minutes
  • Tone: neutral, no labels, no judgment
  • If unsure: Neutral is a valid choice
  • If a question feels sensitive: use Skip
This is not a definitive diagnosis. It provides an approximate profile.
Privacy and data (short and clear)

In this version, answers are stored and processed in your browser (sessionStorage). There is no built-in form submission or custom analytics logging.

If this page is served from a website, normal technical server logs (such as IP logs) may still exist like any website. For higher privacy, use your browser's Private/Incognito mode.
